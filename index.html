🔧 Hướng dẫn kết nối Firebase

Để hệ thống hoạt động với dữ liệu thật, bạn cần tạo dự án Firebase và điền thông tin cấu hình vào file index.html. Thực hiện theo các bước sau:

1Vào console.firebase.google.com → Đăng nhập bằng Google → Nhấn "Add project"
2Đặt tên dự án: chibo-system → Tiếp tục → Tắt Google Analytics (không cần) → "Create project"
3Nhấn biểu tượng </> Web → Đặt tên app: chibo-web → Nhấn "Register app" → Copy toàn bộ firebaseConfig
4Trong menu trái: Authentication → Get started → Email/Password → Enable → Save
5Trong menu trái: Firestore Database → Create database → Start in test mode → chọn vùng asia-southeast1 → Done
6Mở file index.html trên GitHub → Sửa phần firebaseConfig → Dán config vừa copy vào → Commit changes
7Tạo tài khoản đầu tiên (Quản trị viên) bằng cách vào Firebase Console → Authentication → Add user → Nhập email + mật khẩu → Sau đó vào Firestore → users collection → Tạo document với id = UID của user vừa tạo, gồm các field: displayName, email, role: "quantri", department, active: true
// Ví dụ firebaseConfig cần điền vào: const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXX", authDomain: "chibo-system.firebaseapp.com", projectId: "chibo-system", storageBucket: "chibo-system.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef" };
